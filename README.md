# capitalManager
个人记账本，创建的用户以及账本信息存储在切片中
#### 涉及基础内容
    函数调用，循环，分支选择，结构体，切片
#### 记账本菜单
-------------------------------个人记账本-----------------------------

                              1: 用户登录
                              2: 创建用户
                              3: 退出记账
请输入《1-3》
#### 用户菜单
-------------------------------个人记账本-----------------------------

                              1: 收支明细
                              2: 登记收入
                              3: 登记支出
                              4: 退出登录
请输入《1-4》
